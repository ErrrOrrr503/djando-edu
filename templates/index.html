О проекте

Это небольшая тестирующая система для заданий по информатике, реализованная на Django в учебных целях.

Система позволяет создавать и решать контесты с задачами.

Возможности

  • Учетная запись учителя, чтобы создавать контесты и задачи. Без входа - только просмотр и решение.
  • Два типа задач: с текстовым ответом и с ответом в виде кода на python.
  • Для оформления текста заданий учитель может использовать html.
  • Автоматическая проверка заданий:
    • Для задач с ответом в виде текста - сравнение с эталоном.
    • Для задач с ответом в виде программы - запуск тестов.
